# トキワテック 人事考課システム

エンジニアの評価・育成を支援する人事考課システム

---

## プロジェクト概要

30名規模のエンジニア組織向けに、透明性・納得性のある評価制度と、自律的なスキル成長を支援するカリキュラム生成機能を提供するシステム。

## 主要機能（予定）

1. **スキル管理** - 82項目のロードマップに基づくスキル達成状況の管理
2. **評価管理** - 年次評価の入力・算出・フィードバック
3. **昇格管理** - 昇格試験のスコアリング・判定
4. **カリキュラム生成** - LLMを活用した個別学習カリキュラムの生成
