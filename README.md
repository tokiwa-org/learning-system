# Learning System

エンジニア育成のためのミッション型学習システム

## 概要

- スキルロードマップに基づく学習カリキュラム
- ミッション形式による実践的な学習体験
- L0（未経験）〜L5（シニア）までの段階的育成

## 技術スタック

- **Frontend**: Astro + React + Tailwind CSS
- **Backend**: Cloudflare Workers (Hono)
- **Database**: Cloudflare D1
- **Knowledge Graph**: Neo4j Aura

## セットアップ

```bash
# 依存関係インストール
npm install
cd ui && npm install

# 開発サーバー起動
npm run dev        # Backend
cd ui && npm run dev  # Frontend
```
