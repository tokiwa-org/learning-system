# 画面設計

本ドキュメントは、人事考課システムの画面構成と画面遷移を定義します。

---

## 1. 画面一覧

### 1.1 社員向け画面

| ID   | 画面名           | 説明                                     |
| ---- | ---------------- | ---------------------------------------- |
| SC01 | ログイン         | ログイン画面                             |
| SC02 | ダッシュボード   | ホーム画面（評価・学習課題・スキル概要） |
| SC05 | 評価結果確認     | 自分の評価結果の確認                     |
| SC06 | スキルマップ     | 自分のスキル達成状況（レーダーチャート） |
| SC07 | ロードマップ詳細 | ロードマップ項目の詳細                   |
| SC08 | 通知一覧         | 通知の一覧表示                           |
| SC09 | プロフィール     | 自分の情報確認                           |

### 1.2 社員向け学習画面

| ID   | 画面名         | 説明                                   |
| ---- | -------------- | -------------------------------------- |
| SC25 | 学習課題一覧   | 割り当てられた学習課題の一覧           |
| SC26 | 学習課題実行   | カリキュラムを実行する画面             |
| SC27 | 学習結果確認   | 完了した学習課題の結果・フィードバック |

※ 評価・学習課題はダッシュボードから直接アクセス可能

### 1.3 上司向け画面

| ID   | 画面名           | 説明                   |
| ---- | ---------------- | ---------------------- |
| SC10 | 部下一覧         | 部下の評価状況一覧     |
| SC11 | 上司評価入力     | 部下の評価入力         |
| SC12 | 評価承認         | 評価の承認・差戻し     |
| SC13 | 部下スキル確認   | 部下のスキル状況確認   |
| SC28 | 上司ダッシュボード | 上司専用のダッシュボード |

### 1.4 HR・管理者向け画面

| ID   | 画面名             | 説明                               |
| ---- | ------------------ | ---------------------------------- |
| SC14 | 管理ダッシュボード | 全体進捗確認                       |
| SC15 | 評価期間管理       | 評価期間のCRUD                     |
| SC16 | 社員管理           | 社員情報の管理                     |
| SC17 | 評価基準管理       | 評価基準の設定                     |
| SC18 | HR承認             | 評価の最終承認・確定               |
| SC19 | 昇格管理           | 昇格候補者の確認・実行             |
| SC20 | レポート           | 評価レポート出力                   |
| SC21 | 監査ログ           | 監査ログ閲覧                       |
| SC22 | シナリオ管理       | 学習用シナリオの作成・管理         |
| SC23 | カリキュラム管理   | 生成されたカリキュラムの確認・承認 |
| SC24 | 学習課題割り当て   | 社員への学習課題配信               |
| SC29 | システム設定       | システム全体の設定管理             |
| SC30 | ヘルプ             | ヘルプ・ドキュメント画面           |
