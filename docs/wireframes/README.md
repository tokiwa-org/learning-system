# ワイヤーフレーム

このフォルダには、Excalidrawで作成した画面ワイヤーフレームを格納します。

## ツール

- **Excalidraw**: https://excalidraw.com/
- **VS Code 拡張**: `pomdtr.excalidraw-editor`

## ファイル命名規則

```
{画面ID}_{画面名}.excalidraw.png
```

例：

- `F01_職級マスタ管理.excalidraw.png`
- `F07_達成状況入力.excalidraw.png`
- `F08_評価シート.excalidraw.png`

## 作成ガイドライン

### カラーパレット

| 用途       | カラー | Hex       |
| ---------- | ------ | --------- |
| プライマリ | Blue   | `#1971c2` |
| セカンダリ | Gray   | `#868e96` |
| アクセント | Green  | `#2f9e44` |
| 警告       | Orange | `#f76707` |
| エラー     | Red    | `#e03131` |
| 背景       | Light  | `#f8f9fa` |

### コンポーネント

- **ヘッダー**: 高さ 60px、背景 #1971c2
- **サイドバー**: 幅 240px（折り畳み時 60px）
- **カード**: 角丸 8px、シャドウあり
- **ボタン**: 角丸 4px、高さ 36px
- **テーブル**: ヘッダー背景 #f1f3f5

## 編集方法

1. VS Code で `.excalidraw.png` ファイルを開く
2. Excalidraw エディタで編集
3. 保存すると PNG として自動エクスポート
