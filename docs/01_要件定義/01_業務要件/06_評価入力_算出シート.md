# 評価入力・算出シート

---

## 入力項目一覧

| 列 | 項目名 | 説明 | 入力/自動 |
|----|--------|------|----------|
| A | 氏名 | 対象者氏名 | 入力 |
| B | 社歴年数 | 入社からの年数 | 入力 |
| C | 現在の職級 | 1〜5 | 入力 |
| D | 現在の号俸 | 1〜40 | 入力 |
| E | 評価スコア | 0〜100点 | 入力 |
| F | 評価ランク | S/A/B/C/D | 自動算出 |
| G | 号俸アップ数 | 評価ランクから算出 | 自動算出 |
| H | 次年度号俸 | D + G | 自動算出 |
| I | 基礎給 | 社歴年数から参照 | 自動算出 |
| J | 職級給 | 職級×次年度号俸から参照 | 自動算出 |
| K | 合計支給額 | I + J | 自動算出 |

---

## 算出ロジック

### F列：評価ランク

```
IF(E >= 90, "S",
   IF(E >= 75, "A",
      IF(E >= 60, "B",
         IF(E >= 45, "C", "D"))))
```

### G列：号俸アップ数

| 評価ランク | 号俸アップ |
|------------|------------|
| S | +5 |
| A | +3 |
| B | +2 |
| C | +1 |
| D | ±0 |

### H列：次年度号俸

```
= D + G
※ 上限40号俸
```

### I列：基礎給

```
= VLOOKUP(B, 基礎給テーブル, 2, FALSE)
```

### J列：職級給

```
= INDEX(職級給テーブル, H, C)
```

### K列：合計支給額

```
= I + J
```
