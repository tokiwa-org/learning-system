# 評価点算出ロジック

本ドキュメントは、人事考課における評価点の算出方法を定義します。

---

## 1. 評価軸と配点

| 評価軸         | 対象項目  | 配点      | 比重     |
| -------------- | --------- | --------- | -------- |
| ① スキル習得度 | No.1〜46  | 50点      | 50%      |
| ② 職能発揮力   | No.52〜73 | 30点      | 30%      |
| ③ 行動・貢献   | No.74〜82 | 20点      | 20%      |
| **合計**       |           | **100点** | **100%** |

---

## 2. スキル習得度（50点満点）

### 算出方法

現職級の**必須項目**に対する達成率で算出。

```
スキル習得度スコア = 達成項目数 / 必須項目数 × 50
```

### 算出例

| 職級 | 必須項目数 | 達成項目数 | 達成率 | スコア |
| ---- | ---------- | ---------- | ------ | ------ |
| L2   | 20項目     | 15項目     | 75%    | 37.5点 |
| L3   | 25項目     | 22項目     | 88%    | 44点   |

### 達成判定基準

| 状態   | 定義                         | カウント |
| ------ | ---------------------------- | -------- |
| 未着手 | まだ取り組んでいない         | ×        |
| 進行中 | 取り組み中だが完了していない | ×        |
| 完了   | 要件を満たし、エビデンスあり | ○        |

---

## 3. 職能発揮力（30点満点）

### 算出方法

No.52〜73の各項目（22項目）を5段階評価し、平均点から算出。

```
職能発揮力スコア = (各項目の評価点の合計 / 項目数) / 5 × 30
```

### 5段階評価基準

| 評価 | 点数 | 意味                               |
| ---- | ---- | ---------------------------------- |
| 5    | 5点  | 期待を大きく上回る（次職級レベル） |
| 4    | 4点  | 期待を上回る                       |
| 3    | 3点  | 期待通り（現職級の標準）           |
| 2    | 2点  | 期待を下回る                       |
| 1    | 1点  | 大幅な改善が必要                   |

### 算出例

| 評価者 | 項目数 | 合計点 | 平均点 | スコア          |
| ------ | ------ | ------ | ------ | --------------- |
| 上司   | 22項目 | 77点   | 3.5    | 3.5/5×30 = 21点 |

---

## 4. 行動・貢献（20点満点）

### 算出方法

No.74〜82の各項目（9項目）を5段階評価。 **自己評価と同僚評価の加重平均**で算出。

```
行動・貢献スコア = 加重平均点 / 5 × 20

加重平均点 = 自己評価平均 × 0.3 + 同僚評価平均 × 0.3 + 上司評価平均 × 0.4
```

### 同僚評価の取り扱い

| 条件              | 処理                   |
| ----------------- | ---------------------- |
| 同僚評価が1件のみ | そのまま使用           |
| 同僚評価が2件以上 | 平均値を使用           |
| 同僚評価が0件     | 自己:上司 = 4:6 で計算 |

### 算出例

| 評価者 | 項目数 | 合計点 | 平均点 |
| ------ | ------ | ------ | ------ |
| 自己   | 9項目  | 31.5点 | 3.5    |
| 同僚A  | 9項目  | 36点   | 4.0    |
| 同僚B  | 9項目  | 34点   | 3.78   |
| 上司   | 9項目  | 33点   | 3.67   |

```
同僚平均 = (4.0 + 3.78) / 2 = 3.89
加重平均 = 3.5×0.3 + 3.89×0.3 + 3.67×0.4 = 1.05 + 1.17 + 1.47 = 3.69
スコア = 3.69 / 5 × 20 = 14.76点
```

---

## 5. 合計点と評価ランク

### 合計点の算出

```
合計点 = スキル習得度 + 職能発揮力 + 行動・貢献
```

### 評価ランク決定テーブル

| 合計点           | 評価ランク | 号俸アップ |
| ---------------- | ---------- | ---------- |
| 90点以上         | S          | +5         |
| 75点以上90点未満 | A          | +3         |
| 60点以上75点未満 | B          | +2         |
| 45点以上60点未満 | C          | +1         |
| 45点未満         | D          | ±0         |

---

## 6. 総合算出例

### 例: 山田太郎（L2、社歴2年目）

**① スキル習得度**

- L2必須項目: 20項目
- 達成項目: 16項目
- スコア: 16/20 × 50 = **40点**

**② 職能発揮力**

- 上司評価合計: 77点（22項目）
- 平均: 3.5
- スコア: 3.5/5 × 30 = **21点**

**③ 行動・貢献**

- 自己平均: 3.5、同僚平均: 3.89、上司平均: 3.67
- 加重平均: 3.69
- スコア: 3.69/5 × 20 = **14.76点**

**合計**

```
40 + 21 + 14.76 = 75.76点 → A評価（+3号俸）
```

**給与計算**

- 現在: L2・6号俸 → 149,600円（職級給）
- 次年度: L2・9号俸 → 156,100円（職級給）
- 基礎給（2年目）: 105,000円
- 合計: 261,100円

## 7. 端数処理

| 計算段階       | 処理                          |
| -------------- | ----------------------------- |
| 各軸のスコア   | 小数点第2位を四捨五入         |
| 合計点         | 小数点第1位を四捨五入（整数） |
| 評価ランク判定 | 整数で判定                    |

## 8. 特殊ケースの取り扱い

### 8.1 入社1年目（新卒・中途）

| 条件                 | 対応                                         |
| -------------------- | -------------------------------------------- |
| 入社6ヶ月未満        | 評価対象外（翌年度から）                     |
| 入社6ヶ月以上1年未満 | 評価実施。ただしスキル習得度は対象期間で按分 |

### 8.2 異動・職級変更があった場合

| ケース         | 対応                                 |
| -------------- | ------------------------------------ |
| 期中に職級変更 | 期末時点の職級で評価                 |
| 期中に部署異動 | 前後の上司がそれぞれ評価、人事が統合 |

### 8.3 休職期間がある場合

| 条件            | 対応                             |
| --------------- | -------------------------------- |
| 3ヶ月未満の休職 | 通常通り評価                     |
| 3ヶ月以上の休職 | 在籍期間で按分、または評価見送り |
