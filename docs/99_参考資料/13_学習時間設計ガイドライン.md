# 学習時間設計ガイドライン

## 概要

本システムでは、社員一人あたり**月20時間**の学習時間を確保することを目標とします。
これは年間240時間に相当し、研修熱心な企業の上位水準です。

---

### 年間研修時間の比較（2024年データ）

| 企業タイプ | 年間時間 | 月換算 |
|-----------|---------|--------|
| 米国平均 | 47時間 | 4時間 |
| 中規模企業 | 53時間 | 4.5時間 |
| 製造業 | 64時間 | 5時間 |
| 政府・軍（中規模） | 87時間 | 7時間 |
| 研修熱心な企業 | 170〜270時間 | 15〜20時間 |
| **本システム目標** | **240時間** | **20時間** |

## 2. 月20時間の時間配分

### 2.1 カテゴリ別配分

| カテゴリ | 時間/月 | 割合 | 内容 |
|---------|---------|------|------|
| **必須研修** | 4時間 | 20% | セキュリティ、コンプライアンス、全社共通 |
| **職級別研修** | 6時間 | 30% | スキルロードマップに沿った学習 |
| **専門スキル** | 6時間 | 30% | 担当業務に関連する技術・知識 |
| **自己啓発** | 4時間 | 20% | 本人選択のカリキュラム |
| **計** | **20時間** | 100% | |

### 2.2 週次スケジュール例

```
┌─────────────────────────────────────────────────────────┐
│ 週次学習スケジュール（例）                              │
├─────────────────────────────────────────────────────────┤
│ 月曜 08:30-09:30  朝の学習時間（1時間）                │
│ 水曜 08:30-09:30  朝の学習時間（1時間）                │
│ 金曜 14:00-17:00  午後の研修枠（3時間）                │
├─────────────────────────────────────────────────────────┤
│ 週合計: 5時間 × 4週 = 20時間/月                        │
└─────────────────────────────────────────────────────────┘
```

### 2.3 別パターン：毎日30分 + 週1回集中

```
┌─────────────────────────────────────────────────────────┐
│ 毎日コツコツパターン                                    │
├─────────────────────────────────────────────────────────┤
│ 毎日 12:30-13:00  昼休み後の30分                       │
│ 金曜 14:00-16:30  午後の集中学習（2.5時間）            │
├─────────────────────────────────────────────────────────┤
│ 週合計: 2.5時間 + 2.5時間 = 5時間                      │
│ 月合計: 5時間 × 4週 = 20時間/月                        │
└─────────────────────────────────────────────────────────┘
```

## 3. カリキュラム設計基準

### 3.1 カリキュラム種別と標準時間

| 種別 | 標準時間 | 構成 | 用途 |
|------|---------|------|------|
| **ショート** | 30分 | LESSON×2 | 概念理解、座学 |
| **スタンダード** | 1.5時間 | LESSON×3 + EXERCISE + QUIZ | 基本スキル習得 |
| **ロング** | 3時間 | LESSON×3 + EXERCISE×2 + QUIZ | 実践スキル習得 |
| **ワークショップ** | 4時間 | EXERCISE×3 + 振り返り | ハンズオン中心 |

### 3.2 アイテム別の標準時間

| アイテムタイプ | 標準時間 | 内容量の目安 |
|---------------|---------|-------------|
| LESSON（座学） | 8〜15分 | 1,500〜2,500文字 |
| EXERCISE（演習） | 15〜30分 | タスク2〜3個 |
| QUIZ（クイズ） | 5〜10分 | 問題5〜10問 |

### 3.3 月あたりのカリキュラム数目安

| カテゴリ | 時間 | ショート | スタンダード | ロング |
|---------|------|---------|-------------|--------|
| 必須研修 | 4時間 | 8本 | 2〜3本 | 1本 |
| 職級別研修 | 6時間 | 12本 | 4本 | 2本 |
| 専門スキル | 6時間 | 12本 | 4本 | 2本 |
| 自己啓発 | 4時間 | 8本 | 2〜3本 | 1本 |
| **月合計** | **20時間** | **40本** | **13本** | **6本** |

## 4. 年間カリキュラム計画

### 4.1 年間必要本数

```
月20時間 × 12ヶ月 = 240時間/年

スタンダード（1.5時間）換算: 約160本/年
```

### 4.2 カテゴリ別の年間本数

| カテゴリ | 年間時間 | 本数（1.5時間換算） |
|---------|---------|-------------------|
| 必須研修 | 48時間 | 約32本 |
| 職級別研修 | 72時間 | 約48本 |
| 専門スキル | 72時間 | 約48本 |
| 自己啓発 | 48時間 | 約32本 |
| **計** | **240時間** | **約160本** |

### 4.3 コンテンツライブラリの規模

全社員が異なるカリキュラムを受講する場合を考慮：

| 項目 | 本数 |
|------|------|
| 必須（全社共通） | 30〜50本 |
| 職級別（L1〜L5 × 職種） | 100〜200本 |
| 専門スキル（部門別） | 200〜500本 |
| 自己啓発（選択式） | 100〜300本 |
| **ライブラリ合計** | **400〜1,000本** |

## 5. 職級別の学習配分

### 5.1 職級別フォーカス

| 職級 | 必須 | 職級別 | 専門 | 自己啓発 | 特徴 |
|------|------|--------|------|---------|------|
| **L1（新人）** | 6h | 8h | 4h | 2h | 基礎固め重視 |
| **L2（一人前）** | 4h | 6h | 6h | 4h | バランス型 |
| **L3（中堅）** | 4h | 4h | 8h | 4h | 専門性深化 |
| **L4（リーダー）** | 4h | 6h | 4h | 6h | マネジメント + 自己研鑽 |
| **L5（エキスパート）** | 2h | 4h | 6h | 8h | 高度専門 + 自己選択 |

### 5.2 L1（新人）の年間学習計画例

```
【1年目：基礎固め期】

Q1（入社〜3ヶ月）: 80時間
├─ 必須研修: 20時間（セキュリティ、コンプライアンス、会社理解）
├─ 職級別: 40時間（ビジネス基礎、ツール操作）
├─ 専門: 15時間（配属部門の基礎）
└─ 自己啓発: 5時間

Q2-Q4: 各月20時間 × 9ヶ月 = 180時間
├─ 必須研修: 36時間
├─ 職級別: 54時間（技術基礎、プロジェクト作法）
├─ 専門: 54時間
└─ 自己啓発: 36時間

年間合計: 260時間
```

## 6. 進捗管理

### 6.1 月次目標達成率

| 達成率 | 評価 | アクション |
|--------|------|-----------|
| 100%以上 | 優秀 | 次月の自己啓発枠を増やす |
| 80〜99% | 良好 | 現状維持 |
| 60〜79% | 要改善 | 上長との1on1で原因確認 |
| 60%未満 | 要対策 | 業務調整、学習時間確保の支援 |

- LESSON: {lesson_count}本（各{lesson_minutes}分）
