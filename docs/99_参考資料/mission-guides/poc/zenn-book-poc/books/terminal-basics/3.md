# ターミナルを起動してみよう

いよいよ実際にターミナルを起動します！

お使いのOSに合わせて、起動方法を確認しましょう。

## Windows の場合

### 方法1: Windows Terminal（推奨）

Windows 11には標準搭載。Windows 10はMicrosoft Storeからインストール。

**起動手順:**

1. `Win` キーを押す（スタートメニューが開く）
2. 「terminal」と入力
3. 「Windows Terminal」をクリック

```
┌─────────────────────────────────┐
│ 🔍 terminal                     │
├─────────────────────────────────┤
│ ⬛ Windows Terminal        ← これ │
│ 📁 コマンドプロンプト            │
│ 🔷 PowerShell                   │
└─────────────────────────────────┘
```

### 方法2: ショートカットキー

`Win` + `X` → `I`（ターミナル）または `A`（管理者として実行）

### 方法3: 右クリックメニュー

フォルダ内で `Shift` + 右クリック → 「ターミナルで開く」

## Mac の場合

### 方法1: Spotlight検索（推奨）

**起動手順:**

1. `Cmd` + `Space` を押す（Spotlightが開く）
2. 「terminal」と入力
3. 「ターミナル.app」をクリック

```
┌─────────────────────────────────┐
│ 🔍 terminal                     │
├─────────────────────────────────┤
│ ⬛ ターミナル              ← これ │
│    アプリケーション              │
└─────────────────────────────────┘
```

### 方法2: Finderから

1. Finder を開く
2. 「アプリケーション」→「ユーティリティ」→「ターミナル」

### 方法3: Launchpad

Launchpad → 「その他」フォルダ → 「ターミナル」

## Linux の場合

### 方法1: ショートカットキー（推奨）

`Ctrl` + `Alt` + `T`

これで多くのLinuxディストリビューションでターミナルが開きます。

### 方法2: アプリメニュー

アプリケーションメニュー → 「ターミナル」または「端末」

## 起動したら見える画面

### Windows Terminal (PowerShell)

```

```

### Mac / Linux (zsh または bash)

```

```

または

```

```

## 画面の見方

起動した画面を見てみましょう。

```
┌─────────────────────────────────────────────────┐
│ ❶ タイトルバー                                   │
├─────────────────────────────────────────────────┤
│                                                 │
│ Last login: Mon Jan 27 09:00:00    ← ❷ 情報表示 │
│                                                 │
│ yourname@MacBook ~ %               ← ❸ プロンプト│
│ █                                  ← ❹ カーソル  │
│                                                 │
│                                                 │
└─────────────────────────────────────────────────┘
```

| 番号 | 名称 | 説明 |
|------|------|------|
| ❶ | タイトルバー | ターミナルの名前、タブ |
| ❷ | 情報表示 | 前回のログイン情報など（無い場合もある） |
| ❸ | プロンプト | 「入力してください」のサイン |
| ❹ | カーソル | ここに文字が入力される |

## 最初の一歩：何か入力してみよう

プロンプトが表示されたら、以下を入力してEnterキーを押してください：

```bash
echo "Hello, Terminal!"
```

**結果:**

```
Hello, Terminal!
```

おめでとうございます！ これがあなたの最初のコマンドです！

### 何が起きたのか

| 入力 | 意味 |
|------|------|
| `echo` | 「表示しろ」というコマンド |
| `"Hello, Terminal!"` | 表示する内容 |

`echo` は「こだま」という意味。入力した内容をそのまま返してくれます。

## よくあるトラブル

### 「コマンドが見つかりません」と表示される

```
zsh: command not found: echoo
```

**原因:** コマンドのスペルミス（上の例では `echo` を `echoo` と入力）

**対処:** 正しいスペルで再入力

### 何も表示されない

**原因:** Enterキーを押し忘れている

**対処:** Enterキーを押す

### 日本語が文字化けする

**原因:** 文字コードの設定の問題

**対処:** 今は気にしなくてOK。英語でコマンドを打てば問題なし

## ターミナルを閉じる方法

### 方法1: コマンドで閉じる

```bash
exit
```

### 方法2: ショートカットキー

- Mac: `Cmd` + `Q`
- Windows: `Alt` + `F4`
- 共通: `Ctrl` + `D`（入力がない状態で）

### 方法3: ウィンドウを閉じる

普通のアプリと同じように × ボタンをクリック
