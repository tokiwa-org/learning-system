# L1向け 10ヶ月カリキュラム（年間200時間）

## 概要

| 項目 | 内容 |
|-----|------|
| 対象レベル | L1（新人）→ L2（一人前）への成長 |
| 総学習時間 | 200時間（20時間 × 10ヶ月） |
| 形式 | ミッション型カリキュラム |
| 連動 | Neo4jロードマップ（82スキル項目） |

---

## 月別詳細

### 月1: 開発環境の支配者となろう（20時間）

```yaml
mission:
  title: "開発環境の支配者となろう"
  roadmap:
    skills:
      - name: "OS"
        category: "IT基本"
        achievement: "シェルスクリプトを書ける"
      - name: "Git/バージョン管理"
        category: "IT基本"
        achievement: "ブランチ戦略・コンフリクト解決ができる"
      - name: "HTML/CSS"
        category: "IT基本"
        achievement: "レスポンシブデザイン・Flexbox/Gridを使える"
```

| # | ステップタイトル | 種別 | 時間 |
|---|----------------|------|------|
| 1 | ターミナルを制覇しよう | LESSON | 2h |
| 2 | シェルスクリプトの魔法を習得しよう | LESSON + EXERCISE | 4h |
| 3 | Gitの時間を操ろう | LESSON | 2h |
| 4 | ブランチ戦争を終結させよう | EXERCISE | 4h |
| 5 | CSSグリッドで世界を構築しよう | LESSON + EXERCISE | 6h |
| 6 | 最終試験：環境構築チャレンジ | QUIZ + EXERCISE | 2h |

**背景ストーリー**:
あなたは新人エンジニアとして配属された。先輩から「まずは開発環境を自分で構築できるようになれ」と言われた。ターミナル、Git、基本的なWeb技術をマスターし、開発環境の支配者となれ。
