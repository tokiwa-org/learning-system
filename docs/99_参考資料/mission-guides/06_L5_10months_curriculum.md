# L5向け 10ヶ月カリキュラム（年間200時間）

## 概要

| 項目 | 内容 |
|-----|------|
| 対象レベル | L5（エキスパート） - 組織変革・業界影響レベル |
| 総学習時間 | 200時間（20時間 × 10ヶ月） |
| 形式 | ミッション型カリキュラム |
| ノイズレベル | HIGH（高） |

## L5の特徴

L5カリキュラムは他のレベルと異なり、以下の特徴があります：

### 1. 文化変革が中心
技術的なスキルよりも、組織文化を変革する能力が重視されます。

### 2. 戦略策定能力
個別のタスク遂行ではなく、組織全体の戦略を策定し推進する能力が求められます。

### 3. 影響範囲の拡大
自チームだけでなく、組織全体、さらには業界全体への影響力を発揮することが期待されます。

### 4. 高いノイズレベル
明確な正解がなく、ステークホルダー間の利害調整や組織政治を乗り越える必要があります。
