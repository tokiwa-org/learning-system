# L0向け 10ヶ月カリキュラム（年間200時間）

## 概要

| 項目 | 内容 |
|-----|------|
| 対象レベル | L0（未経験者）→ L1（新人）への成長 |
| 総学習時間 | 200時間（20時間 × 10ヶ月） |
| 形式 | ミッション型カリキュラム |
| ノイズレベル | MINIMAL（最小限） |

## 月別詳細

### 月1: 初めてのターミナルを起動しよう（20時間）

```yaml
mission:
  title: "初めてのターミナルを起動しよう"
  roadmap:
    skill: "OS"
    category: "IT基本"
    target_level: "L0"
    achievement_criteria: "基本的なコマンド操作ができる"
```

| # | ステップタイトル | 種別 | 時間 |
|---|----------------|------|------|
| 1 | 黒い画面の正体を知ろう | LESSON | 2h |
| 2 | ファイルとフォルダを操ろう | LESSON + EXERCISE | 4h |
| 3 | ファイルの中身を見てみよう | LESSON + EXERCISE | 4h |
| 4 | 検索コマンドをマスターしよう | LESSON + EXERCISE | 4h |
| 5 | 権限の世界を理解しよう | LESSON | 2h |
| 6 | ターミナル卒業試験 | QUIZ + EXERCISE | 4h |

**背景ストーリー**:
IT企業への入社初日。先輩から「まずはターミナルを開いてみて」と言われた。黒い画面が表示されるが、何をすればいいのかわからない...。一歩ずつターミナルの使い方を覚えていこう。
