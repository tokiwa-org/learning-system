## 達成目標

このミッション完了後にできること：

- ソフトウェア品質の基本概念を説明できる
- 品質コストの考え方を理解し、予防の重要性を説明できる
- チェックリストを作成し、活用できる
- テストケースを読み、手順に従ってテストを実行できる
- テスト結果やバグを適切に報告できる
- コードレビューを受ける準備ができる
- レビュー指摘に対して適切に対応し、学びに変えられる
