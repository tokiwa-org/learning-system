## すでにアカウントがある場合

GitHubアカウントをすでに持っている場合は、このセクションをスキップして次に進んでください。

確認方法：
1. https://github.com にアクセス
2. 右上に自分のアイコンが表示されていればログイン済み

---

## アカウント作成手順

### Step 1: GitHubにアクセス

ブラウザで以下のURLを開きます：

```
https://github.com
```

### Step 2: サインアップを開始

画面右上の「Sign up」ボタンをクリックします。

### Step 3: メールアドレスを入力

```
Enter your email → あなたのメールアドレスを入力
```

> 仕事用とプライベート用どちらでもOKです。
> 後から変更も可能です。

### Step 4: パスワードを設定

```
Create a password → 強力なパスワードを設定
```

パスワードの要件：
- 15文字以上、または
- 8文字以上で数字と小文字を含む

### Step 5: ユーザー名を決める

```
Enter a username → ユニークなユーザー名を入力
```

ユーザー名は：
- 他の人と重複不可
- 公開される（URLの一部になる）
- 後から変更可能（ただし影響あり）

例：`taro-yamada`、`devtaro2026`

### Step 6: メールの配信設定

```
Would you like to receive product updates and announcements via email?
```

「y」（受け取る）または「n」（受け取らない）を選択します。

### Step 7: 認証

ボットではないことを確認するためのパズルを解きます。

### Step 8: メール認証

入力したメールアドレスに確認コードが届きます。
コードを入力して認証を完了してください。

## 初期設定（任意）

アカウント作成後、いくつかの質問が表示されることがあります。

### チームサイズ

```
How many team members will be working with you?
```

→ 「Just me」（自分だけ）を選択してOK

### 使用目的

```
What features do you want to use?
```

→ スキップしてもOK

### プラン選択

```
Free または Pro
```

→ 「Continue for free」（無料で続ける）を選択

---

## プロフィール設定

### アイコン（アバター）の設定

1. 右上のデフォルトアイコンをクリック
2. 「Settings」を選択
3. 「Avatar」セクションで画像をアップロード

### 表示名の設定

1. Settings → Public profile
2. 「Name」に表示名を入力

> ユーザー名とは別に、表示用の名前を設定できます。

### 自己紹介（Bio）

1. Settings → Public profile
2. 「Bio」に簡単な自己紹介を書く

例：
```
駆け出しエンジニア🔰 Gitを勉強中です！
```

---

## 二要素認証（2FA）の設定

セキュリティ強化のため、二要素認証を設定することを推奨します。

### 設定手順

1. Settings → Password and authentication
2. 「Two-factor authentication」セクション
3. 「Enable」をクリック
4. 認証アプリ（Google Authenticator等）を使用

> 2FAを設定すると、ログイン時にスマートフォンでの認証が必要になります。

## よくあるトラブル

### 確認メールが届かない

1. 迷惑メールフォルダを確認
2. 数分待ってみる
3. 「Resend verification email」をクリック

### ユーザー名が使えない

```
Username is already taken
```

→ 別のユーザー名を試してください。数字を追加するのがコツです。

### パスワードを忘れた

1. ログイン画面で「Forgot password?」をクリック
2. メールアドレスを入力
3. リセット用のメールが届く
