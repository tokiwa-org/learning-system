## 達成目標

このミッション完了後にできること：

- `git init` でリポジトリを作成
- `git add`, `git commit` で変更を記録
- `git push`, `git pull` でリモートと同期
- `git log`, `git diff` で変更履歴を確認
- `git checkout`, `git reset` で変更を取り消し
