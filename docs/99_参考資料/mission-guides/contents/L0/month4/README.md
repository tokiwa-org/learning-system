## 達成目標

このミッション完了後にできること：

- インターネットの基本的な仕組みを説明できる
- TCP/IPモデルの4つの層を理解している
- IPアドレスの仕組み（IPv4、サブネット、プライベートIP）を説明できる
- DNSの名前解決の流れを説明できる
- HTTPのリクエスト/レスポンスモデルを理解している
- `ping`、`nslookup`、`curl`、`traceroute` などのコマンドを使える
