## 用語一覧

### アドレス関連

| 用語 | 読み方 | 意味 |
|------|--------|------|
| **IPアドレス** | アイピーアドレス | コンピュータの住所（例: 192.168.1.1） |
| **MACアドレス** | マックアドレス | ネットワーク機器固有の番号（製造時に決まる） |
| **ポート番号** | ポートばんごう | アプリケーションを識別する番号（例: 80, 443） |
| **ドメイン名** | ドメインめい | 人間が読みやすい名前（例: www.example.com） |

---

### プロトコル関連

| 用語 | 読み方 | 意味 |
|------|--------|------|
| **TCP** | ティーシーピー | データを確実に届けるプロトコル |
| **UDP** | ユーディーピー | データを素早く届けるプロトコル |
| **IP** | アイピー | データの宛先を指定するプロトコル |
| **HTTP** | エイチティーティーピー | Webページの通信プロトコル |
| **HTTPS** | エイチティーティーピーエス | 暗号化されたHTTP |
| **DNS** | ディーエヌエス | ドメイン名をIPアドレスに変換する仕組み |

---

### 機器関連

| 用語 | 読み方 | 意味 |
|------|--------|------|
| **ルーター** | ルーター | ネットワーク間でデータを転送する機器（郵便局） |
| **スイッチ** | スイッチ | LAN内でデータを振り分ける機器（集合ポスト） |
| **サーバー** | サーバー | サービスを提供するコンピュータ |
| **クライアント** | クライアント | サービスを利用するコンピュータ |

---

### ネットワーク関連

| 用語 | 読み方 | 意味 |
|------|--------|------|
| **LAN** | ラン | 狭い範囲のネットワーク（家庭、オフィス） |
| **WAN** | ワン | 広い範囲のネットワーク（都市間、国間） |
| **パケット** | パケット | データを小さく分割した単位 |
| **帯域幅（バンドワイズ）** | たいいきはば | 一度に送れるデータ量（道路の車線数に相当） |
| **レイテンシ** | レイテンシ | データが届くまでの遅延時間 |
| **ISP** | アイエスピー | インターネット接続サービスの提供会社 |

---

### 知っておくと便利な用語

| 用語 | 読み方 | 意味 |
|------|--------|------|
| **ファイアウォール** | ファイアウォール | 不正な通信を遮断する仕組み |
| **NAT** | ナット | プライベートIPとグローバルIPを変換する仕組み |
| **DHCP** | ディーエイチシーピー | IPアドレスを自動で割り当てる仕組み |
| **VPN** | ブイピーエヌ | 安全な通信トンネルを作る技術 |

## 覚え方のコツ

### 郵便の例えで整理

```
IPアドレス     = 住所
ポート番号     = 部屋番号
パケット       = 手紙
ルーター       = 郵便局
TCP            = 書留（届いたか確認）
UDP            = 普通郵便（確認なし）
DNS            = 電話帳（名前→住所を調べる）
HTTP           = 手紙の書き方ルール
帯域幅         = 道路の車線数
```

### 今は全部覚えなくてOK

これからのステップで、それぞれの用語を実際に使いながら学びます。「あ、この用語、見たことある！」という状態になれば十分です。
