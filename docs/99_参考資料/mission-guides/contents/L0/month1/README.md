## 達成目標

このミッション完了後にできること：

- `pwd`, `ls`, `cd` で現在地の確認と移動
- `mkdir`, `touch`, `rm` でファイル・フォルダの作成・削除
- `cat`, `head`, `tail`, `less` でファイル内容の確認
- `find`, `grep` でファイル・内容の検索
- `chmod` で権限の基本操作
