> 「やばい...このままだと金曜の締め切りに間に合わないかも...」

## 最も大切なこと: 早期報告

### 遅れの報告は早いほど良い

```
遅れに気づくタイミングと対応の違い:

水曜日に気づいた場合:
→ 残り2日。計画変更や支援を得る時間がある

木曜日に気づいた場合:
→ 残り1日。なんとか間に合わせるか、延期交渉

金曜日（当日）に気づいた場合:
→ もう手遅れ。信頼を大きく損なう
```

| 報告のタイミング | 影響 |
|-----------------|------|
| 早期（1〜2日前） | 対策の選択肢が多い。信頼を維持できる |
| 直前（当日） | 対策が限られる。信頼が下がる |
| 報告しない | 最悪のパターン。信頼を大きく失う |

> 先輩「『遅れそう』の報告は、悪い知らせじゃない。チームが助けられるチャンスなんだ」

## 遅れの原因を分析する

### よくある原因と対策

| 原因 | 対策 |
|------|------|
| 見積もりが甘かった | 作業分解をやり直し、残り時間を再計算 |
| 仕様が不明確だった | 仕様を確認し、スコープを明確にする |
| 技術的な問題にハマった | 先輩に相談して解決策を見つける |
| 割り込み作業が多かった | 割り込みを減らす、または作業時間を確保 |
| 依存先の遅れ | 依存先の状況を確認し、代替策を検討 |
| 体調不良・集中力低下 | 無理せず上司に相談 |

### 原因分析のテンプレート

```
【遅れの報告】

■ 何が遅れているか:
  トップページのCSS実装

■ どれくらい遅れているか:
  計画より1日遅れ（完了予定: 水曜 → 実際: 木曜以降）

■ 原因:
  デザインの細部（アニメーション部分）の実装が
  想定より複雑だった

■ 影響:
  このままだとレスポンシブ対応とテストの時間が不足する
```

---

## 4つの対処法

### 対処法1: スコープを調整する

**やる範囲を減らして、重要な部分を確実に完了させる**

```
例:
全ページのアニメーションを実装する予定だったが、
トップページだけに絞る。他のページは次のフェーズで対応。

Before: アニメーション全ページ対応（3日）
After:  アニメーションはトップページのみ（1日）
        → 残り2日で他の作業を進められる
```

### 対処法2: 支援を依頼する

**チームメンバーに手伝ってもらう**

```
例:
先輩にCSS実装の一部をお願いする。

「先輩、ナビゲーションのCSS実装を手伝っていただけませんか？
 トップページのCSSに想定以上に時間がかかっていて、
 このままだと金曜の締め切りに間に合わない可能性があります」
```

### 対処法3: 作業を効率化する

**やり方を工夫して時間を短縮する**

```
例:
- CSSフレームワーク（Bootstrap等）を活用する
- 既存のテンプレートを流用する
- 完璧を目指さず、まず動くものを作る
```

### 対処法4: 期限を再調整する

**最終手段: 締め切りの延長を相談する**

```
相談の仕方:

× 「間に合いません」（結論だけ）
○ 「現状の報告と対策の相談があります。
    ○○の原因で△△が遅れています。
    □□の対策を取れば、月曜日までに完了できます。
    スケジュールの調整をご相談させてください」
```

## 報告・相談のフォーマット

### 上司への相談テンプレート

```
【遅れの報告と対策案】

■ 現状:
  トップページのCSS実装が計画より1日遅れています。
  全体の進捗は40%で、計画では50%完了の予定でした。

■ 原因:
  アニメーション部分の実装が想定より複雑でした。

■ 影響:
  このままだと、金曜日のテストが十分にできない可能性があります。

■ 対策案（3つ）:
  案1: アニメーションを簡易版にする → 1日短縮可能
  案2: ナビゲーションのCSS実装を先輩にお願いする → 1日短縮可能
  案3: 締め切りを月曜日に延長する → すべて予定通り実装可能

■ お願いしたいこと:
  案1が現実的だと考えていますが、ご意見をいただけますか？
```

### 報告の3原則

| 原則 | 説明 |
|------|------|
| **事実を伝える** | 感情ではなく、客観的な事実を伝える |
| **対策案を持っていく** | 「どうしましょう？」ではなく、案を複数用意する |
| **自分の意見を添える** | 「自分としては案1が良いと思います」 |

## やってはいけないこと

### 遅れ対応のアンチパターン

| アンチパターン | なぜダメか | 正しい対応 |
|---------------|-----------|-----------|
| 黙って残業で取り戻す | 品質低下、体調悪化の原因 | 報告して計画を調整 |
| 報告しない | 問題が大きくなる | 早期に報告する |
| 「なんとかなります」 | 根拠がないと信頼を損なう | 具体的な対策を示す |
| 他人のせいにする | 信頼を失う | 事実を客観的に伝える |
| パニックになる | 冷静な判断ができなくなる | 深呼吸して整理する |

> 先輩「『遅れています、どうしましょう？』は50点。『遅れています、こう対処したいです』が100点だよ」
