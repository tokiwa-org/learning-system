## 達成目標

このミッション完了後にできること：

- タスク管理ツールを使ってチケットを作成・管理できる
- カンバンボードでタスクの状態を可視化できる
- 重要度と緊急度を基準にタスクの優先順位をつけられる
- 作業時間を見積もり、計画を立てられる
- 締め切りを意識した進捗管理ができる
- 遅延リスクを早期に報告できる
- 複数タスクを効率的に管理できる
- ブロッカー発生時に適切にエスカレーションできる
