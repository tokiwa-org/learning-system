# L0 月10: 新人エンジニアの初仕事を完遂しよう

## 概要

| 項目 | 内容 |
|-----|------|
| 対象 | L0（未経験者） |
| 総時間 | 20時間 |
| スキル | 全L0スキルの統合 |
| 位置づけ | L0カリキュラム最終月（カプストーン） |

---

## ストーリー

> 入社から9ヶ月。ターミナル、Git、データベース、ネットワーク、HTML/CSS、コミュニケーション、タスク管理、品質管理、ドキュメント作成――たくさんのことを学んできた。
>
> そんなある日、先輩から声がかかった。
>
> 「この仕事、一人でやってみて」
>
> これまで学んだことを全て活かして、初めての一人仕事を完遂させよう。

---

## ミッション内容

先輩から依頼された仕事：**「社内ツールの紹介ページを作成し、GitHubで公開する」**

具体的な作業内容：

1. データベースから社内ツールの情報を取得してまとめる
2. HTML/CSSで紹介ページを作成する
3. Gitで管理してGitHubにプッシュする
4. 作業の進捗を日報で報告する
5. 完了報告書を書く

## 達成目標

このミッション完了後にできること：

- 依頼された仕事の要件を正しく理解し、作業計画を立てられる
- 必要な環境を自分で構築できる
- データベースからデータを取得・分析できる
- HTML/CSSでWebページを作成できる
- Gitでバージョン管理し、GitHubに公開できる
- 日報・週報・完了報告書を作成できる
- セルフレビューで品質を担保できる
- L0カリキュラム全課程を修了し、L1への準備ができる
