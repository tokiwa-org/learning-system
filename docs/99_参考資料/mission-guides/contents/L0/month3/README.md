## 学習の流れ

```
Step 1 (2h)     Step 2 (4h)     Step 3 (4h)
[導入・基礎] → [SELECT基本] → [WHERE絞込]
     ↓              ↓               ↓
Step 4 (4h)     Step 5 (4h)     Step 6 (2h)
[追加・更新]  → [並替・集計] → [卒業試験]
```

---

## 達成目標

このミッション完了後にできること：

- データベースとは何かを説明できる
- `SELECT` でデータを取り出せる
- `WHERE` で条件を指定してデータを絞り込める
- `INSERT`, `UPDATE`, `DELETE` でデータを操作できる
- `ORDER BY`, `LIMIT` で結果を整理できる
- `COUNT`, `SUM`, `AVG` で集計できる
