# L0 月6: チームの一員として行動しよう

## 概要

| 項目 | 内容 |
|-----|------|
| 対象 | L0（未経験者） |
| 総時間 | 20時間 |
| スキル | コミュニケーション / 進捗管理・報告 |

---

## ストーリー

> チームに配属されて1週間。毎日何をすればいいのかわからない。
>
> 先輩にどう報告すればいい？困ったときはどうする？
>
> チームの一員として行動する方法を学ぼう。

## 学習の流れ

```
Step 1 (3h)     Step 2 (4h)     Step 3 (4h)
[報連相基礎] → [日報作成]  →  [質問の仕方]
     ↓              ↓               ↓

[相談方法]  →  [ミーティング] → [総合演習]
```

---

## 達成目標

このミッション完了後にできること：

- 報告・連絡・相談の違いを理解し、適切に使い分けられる
- 結論ファーストで簡潔に伝えられる
- 日報を毎日書ける
- 適切な質問文を組み立てられる
- 困ったときに相談のタイミングと方法を判断できる
- チームミーティングで発言できる
- フィードバックを素直に受け止め、改善に活かせる
