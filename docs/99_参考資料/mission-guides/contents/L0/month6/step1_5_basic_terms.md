## ミーティング関連用語

| 用語 | 読み方 | 意味 |
|------|--------|------|
| アジェンダ | あじぇんだ | 会議の議題・話す内容のリスト |
| 議事録 | ぎじろく | 会議の内容を記録した文書 |
| ファシリテーター | ふぁしりてーたー | 会議の進行役 |
| 1on1 | わんおんわん | 上司と部下の定期的な1対1面談 |
| デイリースタンドアップ | でいりーすたんどあっぷ | 毎日の短い進捗共有ミーティング |
| スプリントレビュー | すぷりんとれびゅー | 開発成果を確認する定期的な会議 |
| KPT | けぷと | 振り返りの手法（Keep/Problem/Try） |

### よく聞くフレーズ

```
「アジェンダを事前に共有してください」
→ 会議前に話したい内容をリストにして送ってね

「議事録を取ってもらえますか？」
→ 会議の内容をメモして共有してね

「来週の1on1で相談しましょう」
→ 来週の個別面談で話しましょう
```

## 報告・連絡関連用語

| 用語 | 読み方 | 意味 |
|------|--------|------|
| エスカレーション | えすかれーしょん | 自分では対応できない問題を上位者に報告すること |
| フィードバック | ふぃーどばっく | 行動や成果に対する評価・助言 |
| レビュー | れびゅー | 成果物を確認・評価すること |
| コンセンサス | こんせんさす | 合意・意見の一致 |
| リマインド | りまいんど | 忘れないように再度知らせること |
| FYI | えふわいあい | 参考情報です（For Your Information） |
| ASAP | えーえすえーぴー | できるだけ早く（As Soon As Possible） |

### よく聞くフレーズ

```
「これはエスカレーションが必要です」
→ 自分では解決できないので上の人に報告しよう

「フィードバックをお願いします」
→ 自分の仕事について意見をください

「FYIで共有します」
→ 参考までにお伝えします（対応は不要）
```

## 業務進行関連用語

| 用語 | 読み方 | 意味 |
|------|--------|------|
| タスク | たすく | やるべき作業の単位 |
| マイルストーン | まいるすとーん | プロジェクトの重要な節目 |
| デッドライン | でっどらいん | 最終期限 |
| 優先度 | ゆうせんど | タスクの重要性の順番 |
| ブロッカー | ぶろっかー | 作業を止めている障害 |
| ペンディング | ぺんでぃんぐ | 保留中、一時中断 |
| アクションアイテム | あくしょんあいてむ | 具体的にやるべきこと |

### よく聞くフレーズ

```
「ブロッカーはありますか？」
→ 作業を止めている問題はありますか？

「この件はペンディングにしましょう」
→ この件はいったん保留にしましょう

「アクションアイテムを確認しましょう」
→ 誰が何をいつまでにやるか確認しましょう
```
