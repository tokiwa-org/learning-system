## 達成目標

このミッション完了後にできること：

- HTMLの基本構造を理解し、ページを作成できる
- 見出し、段落、リスト、テーブルなどの基本タグを使える
- リンクや画像を配置し、複数ページのサイトを作れる
- CSSでフォント、色、余白を設定できる
- Flexboxを使ったレイアウトができる
- レスポンシブデザインの基本を理解している
- 自己紹介ページを一から作成できる
