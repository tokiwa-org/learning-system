## L0からの成長マップ

| スキル | L0で学んだこと | L1で学ぶこと |
|--------|---------------|-------------|
| OS | 基本コマンド（ls, cd, mkdir） | シェルスクリプト、プロセス管理、環境構築 |
| Git | add, commit, push, pull | ブランチ戦略、コンフリクト解決、PR/レビュー |
| HTML/CSS | 基本構造、Flexbox | CSS Grid、アニメーション、Sass/SCSS |

---

## 達成目標

このミッション完了後にできること：

- `xargs`, `sed`, `awk` 等の上級コマンドを使いこなせる
- パイプとリダイレクトで複雑なデータ処理ができる
- シェルスクリプトで作業を自動化できる
- ブランチ戦略を理解し、コンフリクトを解決できる
- Pull Requestを作成し、コードレビューに参加できる
- CSS Gridでモダンなレイアウトを構築できる
- Sass/SCSSで効率的なCSS開発ができる
- 開発環境を自分でゼロから構築できる
