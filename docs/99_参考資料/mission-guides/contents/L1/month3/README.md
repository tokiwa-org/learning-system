## 達成目標

このミッション完了後にできること：

- TypeScriptで型安全なコードを書ける
- 配列操作・非同期処理を使いこなせる
- クリーンコードの原則を適用できる
- Pythonの既存コードを読み解き、修正できる
- テスト駆動開発の基本サイクルを回せる
- REST APIクライアントを実装できる
- コードレビューの指摘に対応できる
