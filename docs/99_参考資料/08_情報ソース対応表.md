## 実装時の注意事項

### AWS Prescriptive Guidanceの取り扱い

```python
# PGコンテンツの検索フロー
async def search_prescriptive_guidance(topic: str):
    # Step 1: AWS Knowledge MCPで検索
    result = await aws_knowledge_mcp.search(topic)
    
    if is_sufficient(result):
        return result
    
    # Step 2: 不足時はWeb検索で補完
    web_result = await web_search(
        query=f"{topic} site:docs.aws.amazon.com/prescriptive-guidance",
        trusted_domains=["docs.aws.amazon.com"]
    )
    
    return merge_results(result, web_result)
```

### 設計思想系の特別対応

DDD、クリーンアーキテクチャ、CQRS等の設計思想は、AWS Knowledge MCPでの取得が不安定な場合があるため、以下のフォールバック戦略を推奨：

1. AWS Knowledge MCP → Hexagonal Architecture, Microservices
2. Web検索 → docs.aws.amazon.com/prescriptive-guidance
3. 補助ソース → martinfowler.com, microservices.io
