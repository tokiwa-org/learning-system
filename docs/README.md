## 設計の考え方

### Single Source of Truth: オントロジー

本システムの設計は **Neo4j オントロジー** を唯一の信頼できるソース（Single
Source of Truth）として管理しています。

```
┌─────────────────────────────────────────────────────────────┐
│                    Neo4j オントロジー                        │
│  ┌─────────┐ ┌────────┐ ┌──────────┐ ┌────────┐            │
│  │ Entity  │ │ Screen │ │ API     │ │Workflow│            │
│  │ + Attr  │ │        │ │Endpoint │ │ Status │            │
│  └────┬────┘ └───┬────┘ └────┬─────┘ └───┬────┘            │
└───────│──────────│───────────│───────────│─────────────────┘
        │          │           │           │
        ▼          ▼           ▼           ▼
┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│ D1 Table │ │ Figma    │ │ API      │ │ State    │
│ Schema   │ │ Prompts  │ │ Design   │ │ Machine  │
└──────────┘ └──────────┘ └──────────┘ └──────────┘
```

### オントロジーへのアクセス

**接続情報:**

- URI: `bolt://localhost:37687`
- User: `neo4j`
- Password: `workspwd123`

**主要なクエリ:**

```cypher
-- エンティティ一覧
MATCH (e:Entity) RETURN e.name ORDER BY e.name

-- 画面一覧
MATCH (s:Screen) RETURN s.code, s.name ORDER BY s.code

-- API一覧
MATCH (api:APIEndpoint) RETURN api.method, api.path ORDER BY api.path

-- ワークフロー状態遷移
MATCH (s1:WorkflowStatus)-[t:CAN_TRANSITION_TO]->(s2:WorkflowStatus)
RETURN s1.code, t.action, s2.code
```
